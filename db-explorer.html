DB Explorer

DB Explorer 使い方ガイド

Claude Code で構築した18個の SQLite データベースを、開発やデータ分析に活用するための手順です。

全体フロー

1 DB Explorer でスキーマを確認する

左サイドバーからデータベースを選択し、Schema タブでテーブル構造、Queries タブで代表的なクエリの実行結果、Sample Data タブでサンプルレコードを確認します。SQL をクリックするとクリップボードにコピーされます。

2 ターミナルから直接アクセスする

各 DB カードの「コピー」ボタンで sqlite3 ~/projects/... コマンドをコピーし、ターミナルで実行します。対話モードで自由にクエリを試せます。

3 Claude Code から呼び出す

Claude Code のプロンプトでデータベースのパスとテーブル名を指定するだけで、データの読み取り・分析・加工が可能です。下記の具体例を参照してください。

ターミナルからの直接操作

対話モード

# DB を開く
sqlite3 ~/projects/apps/policy-db/data/policy.db

# よく使う設定
.mode column
.headers on

# テーブル一覧
.tables

# スキーマ確認
.schema councils

# クエリ実行
SELECT ministry_name, count(*) FROM councils GROUP BY ministry_name ORDER BY count(*) DESC;

ワンライナーでクエリ実行

# 省庁別委員会数
sqlite3 -header -column ~/projects/apps/policy-db/data/policy.db \
  "SELECT ministry_name, count(*) as cnt FROM councils GROUP BY ministry_name ORDER BY cnt DESC"

# CSV エクスポート
sqlite3 -header -csv ~/projects/apps/policy-db/data/policy.db \
  "SELECT * FROM gta_core_themes ORDER BY council_count DESC" > ~/Desktop/themes.csv

Python から使う

import sqlite3
import pandas as pd

conn = sqlite3.connect(os.path.expanduser("~/projects/apps/policy-db/data/policy.db"))
df = pd.read_sql("SELECT * FROM gta_core_themes ORDER BY council_count DESC", conn)
print(df.to_string())
conn.close()

メタデータの更新

データベースの構造が変わった場合や新しいDBを追加した場合は、以下のスクリプトでメタデータを再生成してください。

cd ~/projects/apps/app-launcher
python3 scripts/export_db_metadata.py

新しいDBを追加するには scripts/export_db_metadata.py の DATABASES リストにエントリを追加し、index.html の databases 配列にも追加します。

注意事項

  • データベースファイルはローカルのみに存在します（.gitignore で除外済み）。Git には含まれません。
  • 書き込み操作を行う場合は、事前にバックアップを取ることを推奨します: cp db.db db_backup_$(date +%Y%m%d).db
  • 複数プロセスから同時に書き込むと SQLite のロック競合が発生する場合があります。Web アプリが起動中のDBへの直接書き込みは注意してください。
  • DB Explorer に表示されるサンプルデータは export_db_metadata.py 実行時点のスナップショットです。最新データはターミナルや Claude Code で直接確認してください。
