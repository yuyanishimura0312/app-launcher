未来学データ・未来学エージェント

一般社団法人ミラツクが約30冊の未来予測・未来学関連書籍を読書会形式で読み解き、個別予測を「カード」として抽出。 18の大項目と10のマップテーマに整理した成果（2018-2020年）。本ページは原データを格納し、ダウンロード可能にする。 Claude Code から /futurology で未来学エージェントを呼び出すと、本データを参照した質疑応答ができる。

未来学データ（格納済）

ミラツクの原データを app-launcher リポジトリに格納し、JSON / テキスト形式でダウンロード可能。 JSON は18大項目の階層構造を完全保持しており、未来学エージェント・分析スクリプト・他のアプリから参照可能。 テキストファイルは人間可読の整形版で、各カードの原文と出典書籍コードを併記。

未来学エージェント（Claude Code スキル）

Claude Code から /futurology コマンドで起動。本ページに格納されている JSON データを読み込んだ上で、未来予測カードに基づく質問応答・分析・引用ができる。 たとえば「2030年代の人口動態に関する予測を整理して」「ロボット関連のカードと出典書籍を一覧化して」といった依頼に対し、 原典のカード本文と書籍コードを示しながら答える。

スキル定義: ~/.claude/skills/futurology/
データソース: ~/projects/apps/app-launcher/data/futurology/futurology_data.json
